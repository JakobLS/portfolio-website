Some projects I've worked on

Cancer Detection Through Multi-Analyte Blood Test

Thrive Earlier Detection launched in early 2019 with $110M of Series A financing. The project piqued my interest; I thought I could provide an improved cancer detection model with their published proteomics and genomics data.

I succeeded: my model improved sensitivity for breast, colorectal and pancreas cancers by 100%, 31% and 21%, respectively, while maintaining specificity at >99%. These three cancer types account for more than four million new cases annually.

I gained deeper understanding of the entire Machine Learning workflow as well as insights into product/market fit as a result of the project.

Assessing Abnormal Chromosomes using Machine Learning

This work was similar to the multi-analyte blood test project, but focused on improving specificity by evaluating abnormal blood chromosome counts. Specificity was indeed improved to 99.8%, slightly higher than before.

The project gave me extensive knowledge of the entire EDA process, including data transformation and feature engineering, as well as model stacking, model tuning and unbiased model evaluation.

Finding Habitable Exoplanets with Deep Learning

I developed a complete pre-processing pipeline and model architecture in order to apply deep learning to analyze light emitted from distant stars. The data was transformed into spectra and fed into convolutional neural networks to predict three target variables: temperature, gravity and metallicity.

The long-term goal of this project was to find habitable exoplanets and this new method decreased the error on earlier machine learning approaches by more than thirty-five times. As a result, I developed a solid understanding of the computer vision workflow.

Predicting Stock Prices

My motives for this project were more selfish; I hoped that an automated stock-trading system for the Stockholm Stock Exchange might help me pay off student loans.

The project was tougher than expected. Still, I learned a lot about the application of deep learning to time series data, as well as real-time data collection techniques.
